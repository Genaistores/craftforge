# CraftForge by Genaistores LLC

Production-ready Next.js 15 + TypeScript landing page (App Router) for **CraftForge by Genaistores LLC**.

## Tech

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui-style components (built with Radix + Tailwind)

## Requirements

- Node.js 18+

## Getting Started

From this directory:

```bash
npm install
npm run dev
```

Then open:

- `http://localhost:3000`

## Build

```bash
npm run build
```

## Notes

- The “Get My Free Audit” CTA opens a modal with an email field (placeholder submission).
- Hero background uses local SVG assets in `public/` to keep the page fast.
